# dill.moe

The hand-built static source for [dill.moe](https://dill.moe), hosted with GitHub Pages.

There is no build step or CMS dependency. Edit the HTML, CSS, or JavaScript, commit the changes, and GitHub Pages serves the updated site.

## Structure

- `index.html` — homepage
- `assets/css/style.css` — shared OLED-black design system
- `assets/js/scripts.js` — mobile navigation and small enhancements
- each page or article lives in its own folder with an `index.html`
- `media/` contains the site's images
